The Command Line For Designers

General Command Line commands

~

Go to local root folder.

cd foldername

Change directory into foldername.

..

Go up/out one directory.

&&

Put two commands in one line and separate it with &&.

pwd

prints working directory

open foldername

Will open finder window for foldername.

mkdir foldername

Create a directory foldername inside current directory.

touch filename.fileext

Creates a file (remember to include the file extension.

say

Will read aloud anything stated after say command.

Pushing a project from

git status

Checking the status of new, modified or deleted files. Followed by git add .

git add .

Adds changes to the stage. Followed by git commit -m "commit"

git commit -m

Writen as git commit -m "commit message goes here"

Followed by pushing remotely

sublime .

Launches current folder in Sublime

Followed by pushing remotely.

cd foldername

changes to subfolder

ls foldername

lists items inside current folder

ls

lists items inside current folder
